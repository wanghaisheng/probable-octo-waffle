# Global Chat

Cross-protocol AI agent discovery platform. A protocol-agnostic directory that aggregates agents across MCP, A2A, agents.txt, ACDP, and 9+ other protocols. Features a searchable directory of 100,000+ agents across 15+ registries and a free agents.txt validator/linter.

## Key Features

- Protocol-agnostic agent discovery across 15+ registries
- Searchable directory of 100,000+ AI agents
- Free agents.txt validator and linter
- MCP server for programmatic access via npm (@global-chat/mcp-server)

## About llms.txt Implementation

Global Chat provides an `llms.txt` file describing the platform's sections, API endpoints, agent integration flow, and machine-readable files for AI agent interoperability.
