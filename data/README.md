# Generated Data

All files in this directory are **auto-generated** by scripts and should not be edited manually.

- `websites.json` is generated by `scripts/generate-websites.ts` from the `.mdx` files in `packages/content/data/websites/`.

To add or update a website, edit the `.mdx` files in `packages/content/data/websites/` instead. See [CONTRIBUTING.md](../.github/CONTRIBUTING.md) for details.
